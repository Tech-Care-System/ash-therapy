# Story 1.1: Project Initialization (v2 - Modern Stack)

## Status
Done

## Story
**As a** Lead Engineer,
**I want** to initialize the project repository with the latest cutting-edge stack (Next.js 16, React 19, Tailwind v4, HeroUI v3),
**so that** we have a future-proof, high-performance foundation for the "Digital Sanctuary" platform.

## Acceptance Criteria
1.  **Monorepo Structure:** Project is initialized using Turborepo and pnpm workspaces.
2.  **Next.js 16 App:** A new Next.js 16 application (App Router) is created in `apps/web`.
3.  **Modern UI Package:** A shared package `packages/ui` is created with **HeroUI v3 (Beta)** and **Tailwind CSS v4**.
4.  **Tailwind v4 Configuration:** No `tailwind.config.js`. Configuration happens via CSS variables and direct imports as per v4 standards.
5.  **Theme Configuration:** The "Sage" theme (Sage Green primary, Clay secondary, Cream background) is configured using CSS variables in the UI package.
6.  **Integration:** `apps/web` successfully consumes the UI package and displays a test page.
7.  **Linting/Formatting:** ESLint and Prettier are configured for the new stack constraints (e.g., React 19 rules).
